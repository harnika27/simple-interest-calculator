# Code of Conduct

## Our Pledge
We pledge to make participation in this project a harassment-free experience for everyone.

## Our Standards
Be respectful, inclusive, and considerate.

## Enforcement Responsibilities
Project maintainers are responsible for enforcing standards.

## Scope
This code of conduct applies within project spaces and public spaces.

## Enforcement
Instances of abusive behavior may be reported to project maintainers.

## Enforcement Guidelines
Warnings, temporary bans, or permanent bans may be applied.

## Attribution
This Code of Conduct is adapted from the Contributor Covenant.
